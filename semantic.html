Section Tag

Section tags can define elements in a document, such as chapters, headings, or any other area of the document with the same theme.

Example: content with the same theme such as articles about cricket can a single "section". A website's home page could be split into sections for the introduction, news items, and contact information.

Article Section

The "artitcle" element holds content that makes sense on its own. "Article" can hold conent such as article, blogs, comments, magazines, etc.

An article tag would help someone using a screen reader understand where the article content (that might containt a combination of text, images, audio, etc.) begings and ends.

The Aside Element

The aside element is used to mark additional information that can enhance element but isn't required in order to understand the main content. This element can be used alongside other elements such as article or section.

Some common uses of the "aside" element are for:

  • Bibliographies
  • Endnotes
  • Comments
  • Pull quotes
  • Editorial sidebars
  • Additional information

Figure and Figcaption

Figure is an element used to encapsulate media such as image, illustation, diagram, code snippet, etc. We use figure element to encapsulate our img tag.

To add a caption to the image, we use figcaption element. The caption will go inside the "figure", which is different to using "p" element to describe the content.

Audio and Attributes

We use the audio element to embed audio content into a document. We also use attributes to the "audio" tag to be able to have any controls.

List of attributes can be found: Mozzila

Video and Embed

Video

We can use the media element for videos or pictures, but by using a video element, it makes it clear that the a developer is attempting to display a video to the user.

We use the video element to embed video content into a document. We also use attributes to the "video" tag to be able to have any controls.

Some attributes that can alter a video playback incl: controls, autoplay, loop, etc.

Embed

We can use the embed tag, which can embed any media content including videos, audio files, and gifs from an external source. This also means that websites that have an embed button have some form of media content that can be added to other websites.

Note: Embed is a self-closing tag, unlike video.
